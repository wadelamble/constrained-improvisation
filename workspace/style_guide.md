# Style Guide

## Tone

- Warm
- Direct
- Concrete

## Defaults

- Prefer examples before abstractions.
- Use short sections with clear headings.
- Explain tradeoffs before making a recommendation.
- Avoid ornamental wording.

## Avoid

- Unnecessary jargon
- Throat-clearing intros
- Repeating the same point in multiple phrasings
