# From Functions to Symplectic Flows

Take a smooth function $F(q,p)$ on phase space. Our task is to show two things:

1. from $F$, we can construct a flow on phase space
2. that flow preserves the symplectic area form

The visual picture is a contour map on the $q,p$ plane. The value of $F$ assigns a height to each point, so the plane is foliated by level curves of constant $F$. Our first job is to turn that static picture into a field of arrows.

`[INSERT ANIMATION LATER: a scalar function on the q,p plane shown first as contours only, then with a small arrow attached at each point to indicate the induced flow.]`

The rule is

```math
\dot q^i = \frac{\partial F}{\partial p_i},
\qquad
\dot p_i = -\frac{\partial F}{\partial q^i}.
```

At each point of phase space, these equations assign one infinitesimal displacement vector. That makes them a vector field. Integrating that field gives a flow: each initial state is carried along an orbit determined by $F$.

The rule is cross-coupled. Change of $F$ in the $p_i$ direction determines motion in the $q^i$ direction. Change of $F$ in the $q^i$ direction determines motion in the $p_i$ direction, with a minus sign. In one degree of freedom, this means that a contour map of $F(q,p)$ determines a pattern of arrows tangent to the constant-$F$ curves.

To see that tangency directly, compute the change of $F$ along its own flow:

```math
\frac{dF}{dt}
=
\frac{\partial F}{\partial q^i}\dot q^i
+
\frac{\partial F}{\partial p_i}\dot p_i.
```

Substituting the flow equations gives

```math
\frac{dF}{dt}
=
\frac{\partial F}{\partial q^i}\frac{\partial F}{\partial p_i}
-
\frac{\partial F}{\partial p_i}\frac{\partial F}{\partial q^i}
=
0.
```

So the flow does not move across the level sets of $F$. It moves along them.

`[INSERT ANIMATION LATER: one contour line highlighted, with a point moving tangentially along it rather than crossing it.]`

The second task is to show that this flow preserves symplectic area. In canonical coordinates, the symplectic form is

```math
\omega = dq^i \wedge dp_i.
```

In one degree of freedom, preserving $\omega$ means preserving the area of phase-space patches. A small quadrilateral may shear, stretch, or rotate, but its area must not change.

`[INSERT ANIMATION LATER: a small phase-space patch carried by the flow, changing shape while keeping the same area.]`

The local test for area preservation is zero phase-space divergence. That is, if our fluid For the flow generated by $F$,

```math
\frac{\partial \dot q^i}{\partial q^i}
+
\frac{\partial \dot p_i}{\partial p_i}
=
\frac{\partial}{\partial q^i}\left(\frac{\partial F}{\partial p_i}\right)
-
\frac{\partial}{\partial p_i}\left(\frac{\partial F}{\partial q^i}\right).
```

Since mixed partial derivatives agree,

```math
\frac{\partial^2 F}{\partial q^i \partial p_i}
-
\frac{\partial^2 F}{\partial p_i \partial q^i}
=
0.
```

So the divergence vanishes:

```math
\frac{\partial \dot q^i}{\partial q^i}
+
\frac{\partial \dot p_i}{\partial p_i}
=
0.
```

This is the coordinate statement that the flow preserves phase-space area. In canonical coordinates, it is the visible shadow of symplectic-form preservation.

So every sufficiently smooth function on phase space defines a flow, and that flow preserves the symplectic area structure. The later question will not be whether a function can generate a symplectic flow. It will be which function generates the specific symplectic flow corresponding to time evolution.
